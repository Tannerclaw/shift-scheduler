# Shift Scheduler

A Progressive Web App (PWA) for scheduling concrete finishing crews across multiple shifts.

## Features

- 📅 **Drag-and-drop scheduling** — Assign workers to time slots visually
- 📱 **WhatsApp integration** — Send schedules directly to crews
- 👷 **Worker management** — Add/edit workers, phone numbers, mark availability
- 🌙 **Dark mode** — Easy on the eyes for early morning shifts
- 💾 **Offline capable** — Works without internet (PWA)
- 📊 **Timeline view** — Visual 10-hour rest enforcement

## Quick Start

1. Open `pwa/index.html` in a browser
2. Add workers in the **Roster** tab
3. Create time slots in **Schedule** tab
4. Drag workers to slots
5. Click **WhatsApp** to send schedule

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Service Worker for offline support
- LocalStorage for data persistence
- WhatsApp `wa.me` API for messaging

## Deployment

This is a static PWA. Deploy to:
- GitHub Pages
- Netlify
- Vercel
- Any static host
