📋 Quick Start Guide

1
Add Your Crew

Go to the Roster tab and add your workers. Include names and phone numbers for WhatsApp/Telegram messaging.

2
Create Schedule Slots

Tap "+ Add Slot" to create time slots for your shifts. Set the date and time for each slot.

3
Assign Workers

Drag workers from the Available pool to each time slot. Tap the ✓ button to mark workers as "Out/Sick" if needed.

4
Send Messages

Tap WhatsApp or Telegram buttons to send the schedule to your crew. Messages include date, time, and assigned workers.

5
Track Crew Separately

Use the Crews tab for completely separate teams. Crews data never mixes with Schedule data.

💡 Pro Tips

  • Import Contacts: Use the "Import Contacts" button to load workers from your phone's contacts
  • Dark Mode: Toggle dark/light mode in the header
  • Edit Anytime: Tap on worker names, slot times, or descriptions to edit
  • Clear Schedule: Use the "Clear" button to remove all assignments (keeps slots)
  • Feedback: Tap "Feedback" to send suggestions or report issues

🔐 Account & Security

  • Change your password in Settings
  • Add phone/email for password reset
  • Session times out after 30 minutes of inactivity

Find these instructions anytime in Settings → Help & Instructions
